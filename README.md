# 💇 Smart Salon Management System

A desktop-based Smart Salon Management System developed using Java 21 and JavaFX. The application automates salon operations including appointment booking, customer management, staff management, billing, notifications, feedback handling, and business reporting.

---

## Features

### Customer
- Customer Registration
- Secure Login
- Password Recovery
- Book Appointments
- Select Services & Add-ons
- Appointment History
- View Invoices
- Submit Feedback
- Manage Profile

### Admin
- Dashboard
- Customer Management
- Staff Management
- Appointment Scheduling
- Service Management
- Invoice Generation
- Complaint Management
- Notification Management
- Reports & Analytics
- Global Search

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/SmartSalon-Management-System.git
```

2. Open in IntelliJ IDEA or Eclipse.

3. Install Java 21.

4. Build using Maven.

5. Run the application.
